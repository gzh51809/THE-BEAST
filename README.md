# THE-BEAST
> 野兽派App，基于react + antd-ui  + js

### 项目工具

| Option         | Tools                     |
| -------------- | ------------------------- |
| 开发环境       | nodejs                    |
| UI框架         | antd、fa-iconfont、swiper |
| JavaScript框架 | React.js                  |
| 开发工具       | VSCode                    |
| 前端构建工具   | webpack                   |
| 包管理工具     | npm                       |
| 打包app工具    | DCloud                    |

### 目录

\根目录

├── README.md					   项目说明文档

├── package.json				  Node.js模块描述文件

├── package-lock.json			  模块版本锁定文件

├── .gitignore					  忽略文件目录

├── config/							  配置文件

├── public/							  公共资料

├── scripts/						  npm scripts

└── src/							  源文件



### 运行项目

npm start						端口号为localhost:3000
